About SMS Spam Shield
Project Overview

SMS Spam Shield is an intelligent, multi-category spam detection system that uses machine learning and explainable AI (XAI) to classify SMS messages.

Features
  • Multi-category classification (Spam, Phishing, Promotional, Legitimate)
  • Multiple ML models (LR, NB, SVM, LSTM)
  • Explainable AI with token-level explanations
  • Real-time predictions
  • Interactive visualization
